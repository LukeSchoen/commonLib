# CPrime Language Test

A little 3D demo project for experimenting with [**CPrime**](https://github.com/LukeSchoen/CPrime).

![CPrime benchmark stats](https://raw.githubusercontent.com/LukeSchoen/commonLib/refs/heads/main/media/Demo.PNG)

# CPrime is free and open source.

1. It extends C with major C++ features (classes, templates, operators, constructors ..)

![CPrime Header Syntax](https://raw.githubusercontent.com/LukeSchoen/commonLib/refs/heads/main/media/StringHeader.png)

2. It brings new features while preserving tiny outputs & impressive compilation speed.

![CPrime Header Syntax](https://raw.githubusercontent.com/LukeSchoen/commonLib/refs/heads/main/media/Stats.png)

3. CPrime's compiler bootstraps & CPrime code builds as c++ code in any c++ compiler.

![CPrime Source Syntax](https://raw.githubusercontent.com/LukeSchoen/commonLib/refs/heads/main/media/v3Source.PNG)
